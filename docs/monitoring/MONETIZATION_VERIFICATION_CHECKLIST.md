# Monetization Verification Checklist

- Confirm AdSense loader script is present on monetized pages.
- Confirm all emitted `ins.adsbygoogle` containers have non-empty `data-ad-slot`.
- Confirm cookie consent contract IDs are present and script initializes.
- Confirm no ad placeholders render as empty slot shells.
- Confirm no console errors tied to ads or consent flow.
- Confirm policy/legal links remain accessible and unchanged.
