# GSC Observation Checklist

- Verify latest sitemap is fetched successfully and has no parsing errors.
- Confirm no new "Duplicate without user-selected canonical" spikes.
- Confirm no sudden index coverage drops for migrated URLs.
- Check no warnings for blocked resources affecting rendered HTML.
- Spot-check migrated tool URLs for canonical and rich result eligibility.
- Track changes for 48-72 hours after deploy before resuming migration.
