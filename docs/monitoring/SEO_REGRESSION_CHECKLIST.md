# SEO Regression Checklist

- Check canonical exists exactly once per migrated page.
- Check `og:image` is absolute HTTPS URL.
- Check titles retain ToolBite suffix and migrated tool title pattern.
- Check `manifest` link exists on migrated pages.
- Check hreflang includes both `x-default` and `en` with canonical URL.
- Check sitemap excludes `/index.html`, `search.html`, and Google verification token URL.
- Check required schema types exist for migrated tools.
